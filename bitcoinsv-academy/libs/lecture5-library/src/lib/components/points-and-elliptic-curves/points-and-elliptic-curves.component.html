Points and Elliptic Curves

Let's create a point using base point G from the secp256k1 koblitz curve and then use it to create a public/private key-pair.

Generate base point G

 G = Point.getG(); Result: {{ G.toString() }}

Generate random 32byte hex value

 phex = Random.getRandomBuffer(32); Result: {{ phex.toString('hex') }}

Generate private key buffer

 p = Bn.fromBuffer(this.phex) Result: {{ p.toString('hex') }}

Generate Public key 'P' from P = pG

 P = G.mul(this.p) Result: {{ P.toString() }}
