Big Numbers!

Let's create a big number and then do some basic arithmetic with it.

Get Random Buffer (displayed in hex): 

                    hex = Random.getRandomBuffer(32).toString('hex');


Result: {{ hex_output }}

Create a big number using our above hex value: 

                    bn = Bn.fromBuffer(hex);


Result: {{ bn_output }}

Add 1 in big number format to our above big number 

                    bx = bn.add(new Bn(1));
