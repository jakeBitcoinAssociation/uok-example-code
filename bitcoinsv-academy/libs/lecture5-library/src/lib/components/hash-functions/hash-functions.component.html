Hash Functions!

Let's create some hashes.

Generate a SHA256 hash

 sha256 = Hash.sha256(Random.getRandomBuffer(32)); Result: {{ sha256.toString('hex') }}

Generate a ripemd160 hash

 ripemd160 = Hash.ripemd160(Random.getRandomBuffer(32)); Result: {{ ripemd160.toString('hex') }}

Generate a HASH256 hash

 hash256 = Hash.sha256Sha256(Random.getRandomBuffer(32)); Result: {{ hash256.toString('hex') }}

Generate a HASH160 hash

 hash160 = Hash.sha256Ripemd160(Random.getRandomBuffer(32)); Result: {{ hash160.toString('hex') }}
