Hash Functions!

Let's create some hashes.

Generate a SHA256 hash

 sha256 = Hash.sha256(Buffer.from('BSV is the high entropy information highway')); Result: {{ sha256.toString('hex') }}

Generate a ripemd160 hash

 ripemd160 = Hash.ripemd160(Buffer.from('BSV is the high entropy information highway')); Result: {{ ripemd160.toString('hex') }}

Generate a HASH256 hash

 hash256 = Hash.sha256Sha256(Buffer.from('BSV is the high entropy information highway'));
hash256Test = Hash.sha256(this.sha256) Result: {{ hash256.toString('hex') }}
Result: {{ hash256Test.toString('hex') }}

Generate a HASH160 hash

 hash160 = Hash.sha256Ripemd160(Buffer.from('BSV is the high entropy information highway'));
hash160Test = Hash.ripemd160(this.ripemd160); Result: {{ hash160.toString('hex') }}
Result: {{ hash160.toString('hex') }}
